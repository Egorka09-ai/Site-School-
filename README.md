# Result — школа английского и немецкого языков

Одностраничный сайт частной языковой школы Result.
Тёмная, тёплая, кинематографичная эстетика. Чистый HTML/CSS/JS без сборки.

## Структура
- `index.html` — разметка и контент
- `styles.css` — дизайн-система, типографика, анимации
- `script.js` — интерактив (reveal-анимации, параллакс, счётчики, форма)

## Запуск
Просто откройте `index.html` в браузере. Сборка не нужна.

## Деплой на GitHub Pages
1. Создайте репозиторий и загрузите эти файлы.
2. Settings → Pages → Source: ветка `main`, папка `/ (root)`.
3. Сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.

## Что заменить под себя
- Фото преподавателей и аватары (заглушки в секциях «Преподаватели» и «Отзывы»).
- Контакты, цифры статистики, тексты отзывов.

## Подключение Telegram (заявки приходят вам в личку)
Сайт статический, поэтому форма отправляет заявку через Telegram-бота.
Один раз настройте:

1. В Telegram откройте **@BotFather** → `/newbot` → задайте имя → получите **токен**.
2. Откройте своего бота и нажмите **Start** (`/start`) — иначе он не сможет вам писать.
3. Узнайте свой **chat_id**: напишите боту **@userinfobot**, он пришлёт числовой ID.
4. В файле `script.js` вверху найдите блок `TELEGRAM` и впишите:
   ```js
   const TELEGRAM = {
     BOT_TOKEN: "ваш-токен-из-BotFather",
     CHAT_ID:   "ваш-числовой-id",
   };
   ```

Пока поля пустые — форма просто показывает «спасибо» без отправки.

> Примечание: токен бота виден в коде сайта (так работает любой статический сайт).
> Это нормально для приёма заявок; при необходимости токен всегда можно
> перевыпустить через @BotFather.
